## Mobile Apps

* [Official Pebble Android app](https://rebble.io/apk) *(For use with Rebble Web Services. Go to help.rebble.io for full installation and setup instructions)*

* [Rebble Mobile App](https://github.com/pebble-dev/mobile-app) *Mobile app to interface with Pebble watches, built by Rebble volunteers, WIP*
