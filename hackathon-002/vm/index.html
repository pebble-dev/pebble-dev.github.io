Hackathon #002

Developer VM

There are a few ways to install the Pebble SDK . You'll need Python 2 and a bit of patience, but it's fairly straightforward.
We suggest a native install, but you might find it easier to use a prebuilt virtual machine.



The Rebble Hackathon VM is based on Ubuntu 22.04 and comes with the SDK pre-installed, as well as VS Code and Firefox with some useful bookmarks.

You can download the VM here, it's around 7GB:
Download VM
The sudo password is 'rebble'.
Use your favourite hypervisor to load the .vmdk file. Virtualbox is one free option.
